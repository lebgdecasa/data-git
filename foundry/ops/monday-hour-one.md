# 🗓️ Monday Hour One — Weekly Planning SOP

Run `/mh1` at the start of every week (Monday morning or Sunday night). One hour of planning
saves 10–12 hours of drift. Plans land in `ops/weeks/YYYY-Wnn.md`.

## The ritual (≤60 min)

1. **Look Back (15 min)** — last week's plan: done / not done / why. Unanswered emails & messages.
   Name last week's #1 constraint → adopt ONE process fix to prevent recurrence.
2. **Look Forward (10 min)** — next 1–3 months: active decisions (`embassy/decisions.md`),
   pipeline follow-ups (`ops/pipeline.md`), upcoming deadlines. Pull this week's share forward.
3. **Prioritize (10 min)** — force-rank. Top 1–3 = PRIORITIES. Schedule other stuff around
   priorities, never priorities around other stuff.
4. **Schedule (15 min)** — named blocks only ("Send 20 cold DMs to X", never "project work").
   Realistic: include meals, breaks, fixed commitments. Email/admin gets contained slots.
5. **Overflow (5 min)** — explicit ad-hoc buffer blocks. Tight constraints compress work.
6. **Color code (5 min)** — recurring meetings 🔵 blue · admin 🔴 red · one-off meetings 🟡 yellow ·
   priorities/projects 🟢 green · breaks 🟣 purple · personal 🟠 orange.
