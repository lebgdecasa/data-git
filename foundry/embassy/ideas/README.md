# 💡 Idea Dossiers

One file per scored idea, produced by `/score <idea>` — scorecard at top, full battle plan below.

Statuses: `scored` → `testing` → `active` / `killed`. The board checks this directory before
scoring anything new (no re-scoring from scratch; diff against the prior dossier).

Killed ideas stay here. A graveyard with reasons is worth more than a clean folder.
